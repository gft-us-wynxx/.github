# GFT US · Wynxx

**Wynxx** is GFT's AI‑assisted software engineering platform. It helps teams
understand, document, test, fix, and modernize code — across web, cloud, and
mainframe stacks — through a web application and a VS Code extension.

This organization hosts the **public sample applications and reference
archetypes** used to demonstrate what Wynxx can do.

## 📚 Learn about Wynxx

- **Documentation** → https://docs.wynxx.app
